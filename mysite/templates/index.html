활용기술




OCR

지정한 영역 내의 글자를 인식하여 텍스트로 전환하는
인공지능 기술입니다.

변환된 텍스트는 새로운 결과 데이터를 도출하기 위한
입력데이터로 사용됩니다.

CMNAI는 작성된 손글씨를 이미지로 저장함과 동시에
OCR기술을 활용해 텍스르로 변환합니다.

KoBERT

BERT모델이란 사람의 말에 대한 문맥을 학습한
자연어 처리 딥러닝 모델입니다.

KoBERT는 SKTBrain에서 공개한 기계번역 모델로
기존 BERT모델에 한국어 데이터를 학습시킨 모델입니다.

이를 통해 CMNAI는 일기에 대한 감정을 도출해
감정달력을 구현합니다.

Web Security

JavaScript 오픈소스 머신러닝 라이브러리로 모델의
관절 정보를 손가락에 매핑하여 손동작을
실시간으로 인식합니다.

CMNAI는 사용자의 손동작을 토대로 여러 활동을
즐길 수 있습니다.
